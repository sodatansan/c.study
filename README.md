# c.study
# C言語学習記録

新・明解C言語（柴田望洋）を使った学習の記録。

## ファイル一覧
- matrix_multiply.c：4×3行列と3×4行列の積を計算する（配列）
- increment_data.c：入力した日付の次の日付を算出（配列、関数、ポインタ）
- bubble_sort.c：任意で入力した数字を昇順にバブルソートする（配列、関数、ポインタ）
- selection_sort.c：任意で入力した数字を昇順に選択ソートする（配列、関数、ポインタ）
- insertion_sort.c：任意で入力した数字を昇順に挿入ソートする（配列、関数、ポインタ）
- shell_sort.c：任意で入力した数字を昇順にシェルソートする（配列、関数、ポインタ）
- quick_sort.c：任意で入力した数字を昇順にクイックソートする（配列、関数、ポインタ、再帰）
- merge_sort.c：任意で入力した数字を昇順にマージソートする（配列、関数、ポインタ、再帰）
- heap_sort.c：任意で入力した数字を昇順にヒープソートする（配列、関数、ポインタ）
- student_sort.c：学生の名前・身長・体重を入力し、任意の項目で昇順ソートする（構造体、ポインタ）
